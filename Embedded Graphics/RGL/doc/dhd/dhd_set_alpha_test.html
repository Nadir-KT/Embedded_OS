dhd_set_alpha_test
Blendunit and Framebuffer Functions | Send Feedback
Summary

Select alpha test.

Syntax
DHD_API_INLINE void dhd_set_alpha_test(dhd_handle_t a_device, dhd_compare_op_t a_compare, dhd_uint8_t a_ref);
Description

Fragments can be discarded based on their alpha value. If the current fragment does not pass the specified test compared to the reference value it will be discarded.

Parameters
Parameters 
Description 
a_device 
Driver context handle (as returned by dhd_init) 
a_compare 
Alpha compare function (see dhd_compare_op_t) 
a_ref 
Reference value to compare against
