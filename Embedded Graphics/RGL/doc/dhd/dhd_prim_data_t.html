dhd_prim_data_t
Structs, Records, Enums | Send Feedback
Summary

Struct defining a single primitive.

Syntax
typedef struct tagdhd_prim_data {
  dhd_uint32_t m_edgecontrol;
  dhd_uint16_t m_xmin, m_xmax;
  dhd_uint16_t m_ymin, m_ymax;
  dhd_uint16_t m_ystart;
  dhd_uint8_t m_stripe_width;
  dhd_uint8_t m_stripe_offset;
} dhd_prim_data_t;
Description

Edge information is stored separately. Stripe information is optional and will be used only when E_DHD_DRAW_STRIPE is set.
