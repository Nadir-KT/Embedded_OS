User Mode API

 

The user mode driver (UMD) is directly linked to an application. The user mode interface should provide maximum performance. Functions are called frequently by the front end and should be very small and inlined whenever possible. 

Each application can instance as many user mode contexts as it wants (usually one per rendering thread).
