dhd_util_color_lerp
State Utility Functions | See Also | Send Feedback
Summary

Quickly set colorization to interpolate.

Syntax
void dhd_util_color_lerp(dhd_handle_t a_device, dhd_col_input_t a_source1, dhd_col_input_t a_source2, dhd_col_input_t a_factor);
Description

To make usage easier simple and common colorization usecases can be set up using the utility functions. When more complexity is required use the full color unit control functions (see below). 

This function programms the color unit to do a linear interpolation between two inputs based on a third input. The operation is carried out on all (RGB and alpha) channels. The alpha channel from a_factor is used as blending weight for the interpolation.

Parameters
Parameters 
Description 
a_device 
Driver context handle (as returned by dhd_init) 
a_source1 
Input to use when factor is 0 (see dhd_col_input_t) 
a_source2 
Input to use when factor is 255 (see dhd_col_input_t) 
a_factor 
Input to use as blending factor (see dhd_col_input_t)
