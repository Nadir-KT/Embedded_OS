dhd_wait_flags_t
Structs, Records, Enums | Send Feedback
Summary

Hardware wait conditions.

Syntax
typedef enum tagdhd_wait_flags {
  E_DHD_WAIT_NONE = 0,
  E_DHD_WAIT_STC = 1<<0,
  E_DHD_WAIT_STR = 1<<1,
  E_DHD_WAIT_STW = 1<<2,
  E_DHD_WAIT_REG = 1<<3,
  E_DHD_WAIT_PSU = 1<<4,
  E_DHD_WAIT_TXI = 1<<5,
  E_DHD_WAIT_TEX = 1<<6,
  E_DHD_WAIT_COL = 1<<7,
  E_DHD_WAIT_BLEND = 1<<8,
  E_DHD_WAIT_CLEAR = 1<<9,
  E_DHD_WAIT_WRITEBACK = E_DHD_WAIT_REG|E_DHD_WAIT_PSU|E_DHD_WAIT_TXI|E_DHD_WAIT_TEX|E_DHD_WAIT_COL|E_DHD_WAIT_BLEND
} dhd_wait_flags_t;
Description

Combine flags to wait for multiple conditions.

Members
Members 
Description 
E_DHD_WAIT_NONE 
Do not wait 
E_DHD_WAIT_STC 
Wait while stream controller is busy 
E_DHD_WAIT_STR 
Wait while stream reader is busy 
E_DHD_WAIT_STW 
Wait while stream writer is busy 
E_DHD_WAIT_REG 
Wait while register file is busy 
E_DHD_WAIT_PSU 
Wait while pixel selection is busy 
E_DHD_WAIT_TXI 
Wait while stages after PSU up to ZSS are busy 
E_DHD_WAIT_TEX 
Wait while stages after ZSS up to TXS are busy 
E_DHD_WAIT_COL 
Wait while stages after TXS up to FBS are busy 
E_DHD_WAIT_BLEND 
Wait while stages after FBS up to FBC are busy 
E_DHD_WAIT_CLEAR 
Wait while clear unit is busy 
E_DHD_WAIT_WRITEBACK 
Wait until all pixels are written (not including clear unit)
