Description

Every possible permutation of alpha, red, green and blue input channels can be selected. The first char is mapped to the output red channel, the second to green and the third to blue. E_DHD_CHANNEL_BAR would map out_red = in_blue, out_green = in_alpha and out_blue = in_red
