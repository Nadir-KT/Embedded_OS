counter for assigning job id's 
m_ringbuffer 
start of ringbuffer video ram (0 if none) 
m_ringpos 
dword write offset in ringbuffer 
m_ringsize 
number of dwords in ringbuffer (power of two) 
m_status 
current GPU status (written by ISR) 
m_last_sync_code 
last processed GPU sync (written by ISR) 
m_last_stop_code 
stop code received by ISR 
m_perfcount_allocmask 
current allocation of performance counters (0 is free) 
m_perfcount_usemask 
currently enabled performance counters 
m_clut_usemask 
currently allocated clut blocks (one bit for each block) 
m_clut_blocksize 
number of clut entries in one clut block (usually clutsize/32) 
m_schedule_iteration 
current job schedule iteration 
m_core_halting 
true if halt request has been set and STATUS_HALTED is to be expected
