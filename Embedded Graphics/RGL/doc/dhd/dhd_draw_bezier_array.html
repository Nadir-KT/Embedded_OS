dhd_draw_bezier_array
Rendering Functions | See Also | Send Feedback
Summary

Render an array of bezier primitives.

Syntax
void dhd_draw_bezier_array(dhd_handle_t a_device, dhd_uint32_t a_primcount, dhd_bezprim_data_t * a_prims, dhd_int32_t a_primpitch, dhd_uint32_t a_edgecount, dhd_drawattribute_flag_t a_attributes);
Description

This function is identical to dhd_draw_array, with the exception that it uses dhd_bezprim_data_t instead of dhd_prim_data_t structures to describe each primitive. This enables the use of bezier offset0 and bezier offset1 data fields per primitive. Something that is required for most beziers. 

 

For an explanation and examples of a_attributes see dhd_draw.

Parameters
Parameters 
Description 
a_device 
Driver context handle (as returned by dhd_init) 
a_primcount 
Number of primitives to draw 
a_prims 
Array of dhd_bezprim_data_t elements (at least a_primcount elements) 
a_primpitch 
Additional spacing between the dhd_prim_data_t structures (usually 0) 
a_edgecount 
Number of edges per primitive 
a_attributes 
Number of attributes to be read from attribute buffers (see dhd_drawattribute_flag_t)
