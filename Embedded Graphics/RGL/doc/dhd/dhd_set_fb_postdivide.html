dhd_set_fb_postdivide
Blendunit and Framebuffer Functions | Send Feedback
Summary

Enable post divide.

Syntax
DHD_API_INLINE void dhd_set_fb_postdivide(dhd_handle_t a_device, dhd_bool_t a_enable);
Description

Blendunit output is converted to non premultiplied format before writeback to the framebuffer.This is done by dividing each color channel by the alpha channel.

Parameters
Parameters 
Description 
a_device 
Driver context handle (as returned by dhd_init) 
a_enable 
Boolean flag. 0 to disable post divide.
