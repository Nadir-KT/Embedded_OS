dhd_get_hw_revision
General Functions | Send Feedback
Summary

Return HW version identifier.

Syntax
dhd_uint32_t dhd_get_hw_revision(dhd_handle_t a_device);
Description

Once a driver context has been established the version of the connected HW instance can be queried using this function.

Parameters
Parameters 
Description 
a_device 
Driver context handle (as returned by dhd_init) 
Returns

A 32bit integer composed from the following bytes 0, major version, minor version and revision (MSB to LSB).
