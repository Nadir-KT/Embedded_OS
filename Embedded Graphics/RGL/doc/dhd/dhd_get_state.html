dhd_get_state
General Functions | See Also | Send Feedback
Summary

Backup the current state for later reuse.

Syntax
dhd_enum_t dhd_get_state(dhd_handle_t a_device, dhd_state_group_t a_groups, dhd_state_t * a_state);
Description

The current state or a specified part of it can be stored into a state storage allocated by dhd_alloc_state.

Parameters
Parameters 
Description 
a_device 
Driver context handle (as returned by dhd_init) 
a_groups 
A combination of dhd_state_group_t elements that specify the parts of the state to be read 
a_state 
Pointer to a state storage (as returned by dhd_alloc_state) 
See Also

dhd_alloc_state, dhd_free_state, dhd_set_state
