D/AVE HD Driver - Internal Documentation
dhd_gpu_init
GPU Call Interface | See Also | Send Feedback
Summary

Initialize kernel mode driver.

Syntax
struct taginit {
  dhd_uint32_t m_instance;
  const dhd_gpu_config_t * m_config;
  dhd_uint8_t m_version_major_out;
  dhd_uint8_t m_version_minor_out;
  dhd_uint8_t m_revision_out;
} m_init;
dhd_uint32_t dhd_gpu_init(dhd_gpu_call_data_t * a_parameters);
Description

Usually called by the OS at boot time. The parameters are passed inside a dhd_gpu_call_data_t (see Kernel Mode Interface for details).

Parameters
Parameters 
Description 
m_instance 
integer index of the HW instance to initialize. 
m_config 
pointer to a dhd_gpu_config_t providing all data for the initialisation 
m_version_major_out 
output field usefull in case of a E_DHD_ERROR_HW_REVISION_MISMATCH 
m_version_minor_out 
output field usefull in case of a E_DHD_ERROR_HW_REVISION_MISMATCH 
m_revision_out 
output field usefull in case of a E_DHD_ERROR_HW_REVISION_MISMATCH 
Returns

E_DHD_OK if successful, E_DHD_ERROR_HW_REVISION_MISMATCH if the HW revision is unsupported by this driver, E_DHD_ERROR_INVALID_ARGUMENT or E_DHD_ERROR_INVALID_PTR if one of the parameters is incorrect or E_DHD_ERROR_NO_MEMORY if the supplied video RAM was insufficient.

Notes

This is the only GPU function that may not be called concurrently by multiple threads and that has to be called using direct dispatch (passing a 0 bridge handle to dhd_gpu_call).
