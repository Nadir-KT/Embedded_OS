D/AVE HD Driver - Internal Documentation
dhd_set_job_checksum_enable
General Functions | Send Feedback
Summary

Enable or disable the job checksum feature.

Syntax
dhd_enum_t dhd_set_job_checksum_enable(dhd_handle_t a_device, dhd_bool_t a_enable);
Description

This feature calculates a checksum of every stream command generated using this drivers' context. The D/AVE HD core will generate an error interrupt whenever the checksum of the stream does not match the expected value, which is checked at each register dump or start of rendering. Errors will currently only be logged as a kernel error message and execution will continue at the next job. This feature will introduce a small performance impact as the checksum needs to be calculated on the CPU and the generated stream will include two additional words per rendering and other relevant operations. The feature will only be available when the driver is compiled with DHD_JOB_CHECKSUM_SUPPORTED which itself leads to a small driver cpu performance hit.

Parameters
Parameters 
Description 
a_device 
Driver context handle (as returned by dhd_init) 
a_enable 
1 to enable the checksum feature, 0 otherwise
