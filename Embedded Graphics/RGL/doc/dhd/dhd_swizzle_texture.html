dhd_swizzle_texture
Conversion Utility Functions | See Also | Send Feedback
Summary

Swizzle texture and store into another buffer.This is an internal function.

Syntax
dhd_enum_t dhd_swizzle_texture(dhd_handle_t a_device, dhd_gpu_ptr_t a_src_address, dhd_uint32_t a_src_width, dhd_uint32_t a_src_height, dhd_uint32_t a_src_pitch, dhd_texel_org_t a_src_format, dhd_gpu_ptr_t a_dst_address, dhd_uint32_t a_dst_height, dhd_uint32_t a_dst_pitch, dhd_texel_org_t a_dst_format, dhd_address_mode_t a_dst_address_mode);
Description

Use dhd_convert_buffer with a swizzled access mode to access this functionality. It will enswizzle one buffer into another one and convert pitch and format on the fly if requested. Destination buffer sizes must satisfy swizzle constraints (divisible by blocksize or square power of two in case of a full swizzle). The resulting buffer can then be used with the specified address mode (eg. E_DHD_ACCESS_SWIZZLE_16).

Parameters
Parameters 
Description 
a_device 
Driver context handle (as returned by dhd_init). 
a_src_address 
Gpu buffer address of the source buffer (see dhd_get_buffer_gpuptr). 
a_src_width 
Source width in pixels. 
a_src_height 
Source height in pixels. 
a_src_pitch 
Source buffer pitch. 
a_src_format 
Source buffer texel format (see dhd_texel_org_t). 
a_dst_address 
Gpu buffer address of the destination buffer (see dhd_get_buffer_gpuptr). 
a_dst_height 
Destination height in pixels. 
a_dst_pitch 
Target pitch. 
a_dst_format 
Target texel format (see dhd_texel_org_t). 
a_dst_address_mode 
Desired swizzle (E_DHD_ACCESS_SWIZZLE_4, E_DHD_ACCESS_SWIZZLE_16 or E_DHD_ACCESS_SWIZZLE_FULL). 
a_dst_width 
Destination width in pixels. 
Returns

E_DHD_OK if successful, E_DHD_ERROR_INVALID_ARGUMENT in case of an invalid input parameters or E_DHD_ERROR_INVALID_PTR in case of a misaligned destination pointer

Notes

The framebuffer will be set the match the destination surface. Calling function has to store/restore it if required.
