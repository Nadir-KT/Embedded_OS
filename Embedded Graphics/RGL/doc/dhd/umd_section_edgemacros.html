Edge Macros

The following macros can be used to build edgecontrol flags for the rendering functions dhd_draw , dhd_draw_array etc. 

 

These macros are usually combine with the flags from dhd_edge_flag_t:

Name 
Description 
E_DHD_ENABLE 
Edge is enabled (only enabled edges are considered in any way) 
E_DHD_SHARED 
Edge is shared (not antialiased) 
E_DHD_USE_ENUM 
Edge is used during enumeration (only Bezier curves use non enum edges) 
E_DHD_IEDGE 
Shortcut for 'inner' edge (combination of E_DHD_ENABLE, E_DHD_USE_ENUM and E_DHD_SHARED) 
E_DHD_OEDGE 
Shortcut for 'outer' edge (combination of E_DHD_ENABLE and E_DHD_USE_ENUM) 

 

Example

A triangle with all three edges not antialiased (inner edges) could be specified like this :

Copy Code
DHD_EDGE0( E_DHD_IEDGE ) | DHD_EDGE1( E_DHD_IEDGE ) | DHD_EDGE2( E_DHD_IEDGE )
