D/AVE HD Driver - Internal Documentation
dhd_fill
Rendering Functions | Send Feedback
Summary

Clear a rectangular region.

Syntax
void dhd_fill(dhd_handle_t a_device, dhd_gpu_ptr_t a_address, dhd_uint32_t a_width, dhd_uint32_t a_height, dhd_uint32_t a_pitch, dhd_uint32_t a_bpp, dhd_uint32_t a_data, dhd_uint32_t a_mask);
Description

The hardware has a dedicated clear unit. It can be used to quickly fill a rectangular region with any data word. Clearing will always be executed in sync with rendering explicit barriers (dhd_wait_fill) are no longer necessary.

Parameters
Parameters 
Description 
a_device 
Driver context handle (as returned by dhd_init) 
a_address 
First byte in the target rectangle 
a_width 
Width of the rectangle in pixels 
a_height 
Height of the rectangle in pixels 
a_pitch 
Number of pixels in one line of the target buffer 
a_bpp 
Bytes per pixel 
a_data 
Pixel data to fill into the memory (raw bits must match the target pixel organization). 
a_mask 
One bit per byte write enable. To write all bytes use 0xf.
